# ⚡ LinkCZ

**A fast, minimal and open social platform for sharing thoughts instantly.**

LinkCZ is a lightweight social network built for speed, simplicity, and privacy.  
No heavy frameworks. No complexity. Just pure web tech + Supabase power.

---

## 🚀 What is LinkCZ?

LinkCZ is a simple social platform where users can:
- Create an account instantly
- Post messages in real time
- Browse a global feed
- Stay connected without unnecessary complexity

It is designed as a learning project and a foundation for scalable social apps.

---

## ⚙️ Tech Stack

- HTML / CSS / JavaScript (Vanilla)
- Supabase (Authentication + Database)
- Row Level Security (RLS)

---

## 🔥 Features

- ⚡ Instant authentication (email & password)
- 📝 Create and publish posts
- 🌍 Global live feed
- 🔐 Secure backend with Supabase RLS
- 📦 No frameworks, ultra-lightweight
